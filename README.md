# VU-mètre
Ce montage est basé sur le kit technoLARP. Il permet de contrôler un ruban de leds à partir d'un potentiomètre ou d'un navigateur web.

## Composants
Vous aurez besoin pour monter le VU-mètre
- d’un kit technoLarp
- d’un câble micro-USB
- d’un potentiomètre 10 000 Ohms
- d’un ruban ou d’un anneau de led ws2812b ou neopixel
- Une batterie 18650 et son support
